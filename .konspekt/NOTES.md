# NOTES

Human scratchpad for this project. Append-only, organized by date.

This file is **outside** the konspekt graph: the `note:` / `note "…"` convention
writes here and is **never** reconciled into a `Noteworthy` entity (see
`/spec/data-model/SPEC.md` — `note:` is reserved for exactly this and is not a
konspekt capture verb). Notes are for the human; the graph is for the project.

Date headings appear as notes are added.
