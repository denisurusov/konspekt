# Task: Build the adoption path

The smallest thing to put in front of a new adopter: a `setup/` kit that
scaffolds a `.konspekt/` umbrella (seed instance + operating envelope) into any
project repo and adds a konspekt stanza to its `AGENTS.md`. Distinct from
`task-second-implementer` (find and recruit a person) — this is the on-ramp the
person arrives at. Ongoing: the v1 kit is the first artifact, not the finish.
