# instance

konspekt, eating its own dog food.

This directory holds the live state of *building konspekt*, recorded in konspekt's own (still-forming) format. The project is its own first guinea pig.

The point of dogfooding here is not proof — the author is a non-representative user who tolerates friction others won't. It is to generate *hypotheses*: each time a convention earns its keep, the portable lesson is the underlying need it served, not the mechanic. Those needs are what a second implementer will care about.

Serialization is provisional. For now this is Markdown that mirrors the data model in `../spec/SPEC.md`; the concrete format is an open decision.
